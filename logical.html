C++ - Logical Operators

Learning Objectives

  • To combine multiple conditions using logical operators.
  • To make more complex decisions in programs.
  • To implement conditions in loops, if-else statements, and validations.

Logical Operators Covered

Operators in this document: && (AND), || (OR), ! (NOT)

1. Logical AND (&&)

Detailed definition

The logical AND operator (&&) returns true if both operands are true, otherwise it returns false.

Example

int a = 5;
int b = 10;
if(a > 0 && b > 0) {
    cout << "Both a and b are positive" << endl;
}

2. Logical OR (||)

Detailed definition

The logical OR operator (||) returns true if at least one of the operands is true. Returns false only if both operands are false.

Example

int a = -5;
int b = 10;
if(a > 0 || b > 0) {
    cout << "At least one of a or b is positive" << endl;
}

3. Logical NOT (!)

Detailed definition

The logical NOT operator (!) negates the value of its operand. Returns true if the operand is false, and false if the operand is true.

Example

bool isRaining = false;
if(!isRaining) {
    cout << "It is not raining" << endl;
}

Real-world Example Using All Logical Operators

Login Validation

bool usernameCorrect = true;
bool passwordCorrect = false;

if(usernameCorrect && passwordCorrect) {
    cout << "Login successful" << endl;
}
if(usernameCorrect || passwordCorrect) {
    cout << "Partial information correct" << endl;
}
if(!usernameCorrect) {
    cout << "Username is incorrect" << endl;
}
