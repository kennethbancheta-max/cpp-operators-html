Basic Assignment Operators Covered

Operators in this document: =, +=, -=, *=, /=, %=

1. Simple Assignment (=)

Detailed definition

The simple assignment operator stores the value of the right-hand side (RHS) expression into the left-hand side (LHS) variable. The RHS expression is fully evaluated first; then the resulting value replaces any previous value in the LHS variable. This operator is used to initialize variables and update their values during runtime.

Example

#include <iostream>
using namespace std;

int main() {
    int x;
    x = 10;
    cout << "x = " << x << endl;
    return 0;
}

2. Add and Assign (+=)

Detailed definition

The += operator adds the RHS value to the current value of the LHS variable and assigns the resulting sum back into the LHS variable. It is a shorthand for x = x + expression. The RHS expression is evaluated first and then added to the current LHS value before assignment.

Example

int x = 5;
x += 3;  // x is now 8

Real-world Example Using All Operators

Shopping Cart Scenario

#include <iostream>
#include <string>
using namespace std;

int main() {
    string customer = "John";
    int totalItems = 0;
    int totalPrice = 0;
    int discount = 0;
    int tax = 0;

    totalItems = 5;
    totalPrice = 100;

    totalItems += 3;
    totalPrice += 60;

    totalItems -= 2;
    totalPrice -= 40;

    totalPrice *= 0.9; // 10% discount

    tax = totalPrice;
    tax /= 100;
    tax *= 5;
    totalPrice += tax;

    int boxes = totalItems / 4;
    int leftoverItems = totalItems;
    leftoverItems %= 4;

    cout << "Customer: " << customer << endl;
    cout << "Total Items: " << totalItems << endl;
    cout << "Total Price: $" << totalPrice << endl;
    cout << "Boxes: " << boxes << ", Leftover Items: " << leftoverItems << endl;

    return 0;
}
