C++ - Comparison Operators

Learning Objectives

  • To compare two values using relational operators.
  • To make decisions in the program using boolean outcomes.
  • To combine comparison with logical operators for complex conditions.
  • To use comparisons in loops, conditions, and data validation.

Comparison Operators Covered

Operators in this document: ==, !=, >, <, >=, <=

1. Equal to (==)

Detailed definition

The == operator checks whether two operands are equal. It returns true if they are equal, otherwise false.

Example

int a = 5;
int b = 5;
if(a == b) {
    cout << "a and b are equal" << endl;
}

2. Not equal to (!=)

Detailed definition

The != operator checks whether two operands are not equal. Returns true if they are not equal, otherwise false.

Example

int a = 5;
int b = 3;
if(a != b) {
    cout << "a and b are not equal" << endl;
}

3. Greater than (>)

Detailed definition

The > operator checks if the left operand is greater than the right operand. Returns true if it is, otherwise false.

Example

int a = 7;
int b = 5;
if(a > b) {
    cout << "a is greater than b" << endl;
}

4. Less than (<)

Detailed definition

The < operator checks if the left operand is less than the right operand. Returns true if it is, otherwise false.

Example

int a = 2;
int b = 5;
if(a < b) {
    cout << "a is less than b" << endl;
}

5. Greater than or equal to (>=)

Detailed definition

The >= operator checks if the left operand is greater than or equal to the right operand. Returns true if it is, otherwise false.

Example

int a = 5;
int b = 5;
if(a >= b) {
    cout << "a is greater than or equal to b" << endl;
}

6. Less than or equal to (<=)

Detailed definition

The <= operator checks if the left operand is less than or equal to the right operand. Returns true if it is, otherwise false.

Example

int a = 3;
int b = 5;
if(a <= b) {
    cout << "a is less than or equal to b" << endl;
}

Real-world Example Using All Comparison Operators

Age Validation

int age = 20;
if(age >= 18) {
    cout << "Eligible to vote" << endl;
}
if(age < 18) {
    cout << "Not eligible to vote" << endl;
}
if(age == 20) {
    cout << "Age is exactly 20" << endl;
}
if(age != 25) {
    cout << "Age is not 25" << endl;
}
