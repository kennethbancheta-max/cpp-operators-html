C++ - Increment/Decrement and Bitwise Operators

Learning Objectives

  • To understand pre- and post-increment/decrement operations.
  • To perform bit-level operations using bitwise operators.
  • To manipulate data efficiently using low-level operations.
  • To use these operators in loops, counters, and algorithmic computations.

Increment/Decrement Operators

1. Increment (++)

Increases the value of a variable by 1. Can be used as prefix (++x) or postfix (x++).

int x = 5;
cout << ++x << endl; // prefix: increments before use, outputs 6
cout << x++ << endl; // postfix: increments after use, outputs 6, x becomes 7

2. Decrement (--)

Decreases the value of a variable by 1. Can be used as prefix (--x) or postfix (x--).

int y = 5;
cout << --y << endl; // prefix: decrements before use, outputs 4
cout << y-- << endl; // postfix: decrements after use, outputs 4, y becomes 3

Bitwise Operators

1. AND (&)

Performs bitwise AND between two integers.

int a = 5; // 0101
int b = 3; // 0011
int c = a & b; // 0001 = 1
cout << c << endl;

2. OR (|)

Performs bitwise OR between two integers.

int a = 5; // 0101
int b = 3; // 0011
int c = a | b; // 0111 = 7
cout << c << endl;

3. XOR (^)

Performs bitwise XOR between two integers.

int a = 5; // 0101
int b = 3; // 0011
int c = a ^ b; // 0110 = 6
cout << c << endl;

4. NOT (~)

Performs bitwise NOT (complement) of an integer.

int a = 5; // 0101
int c = ~a; // 1010 (in 2's complement: -6)
cout << c << endl;

5. Left Shift (<<)

Shifts bits of a number to the left, filling with zeros.

int a = 5; // 0101
int c = a << 1; // 1010 = 10
cout << c << endl;

6. Right Shift (>>)

Shifts bits of a number to the right.

int a = 5; // 0101
int c = a >> 1; // 0010 = 2
cout << c << endl;

Real-world Example Using Increment/Decrement and Bitwise Operators

int counter = 0;
counter++; // increment
counter--; // decrement

unsigned char flags = 0b00001100;
flags |= 0b00000001; // set first bit
flags &= ~0b00001000; // clear fourth bit
cout << "Counter: " << counter << endl;
cout << "Flags: " << (int)flags << endl;
